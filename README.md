## 📋 Tổng quan dự án

**Mobifone** là một nền tảng web thương mại điện tử viễn thông toàn diện mã nguồn mở (open source), được phát triển bằng **Laravel 11**, cho phép khách hàng tra cứu, đăng ký các gói cước di động, mua SIM số trực tuyến, và quản lý các dịch vụ viễn thông Mobifone. Dự án tích hợp đầy đủ hệ thống quản trị (Admin Panel) với phân quyền chi tiết, hỗ trợ quy trình nghiệp vụ từ tiếp thị, bán hàng đến chăm sóc khách hàng.

**Mục tiêu:** Xây dựng một hệ thống quản lý dịch vụ viễn thông tập trung, thay thế quy trình thủ công bằng nền tảng số hiện đại, dễ sử dụng và có khả năng mở rộng cao.

## ⚙️ Hướng dẫn cài đặt

### Yêu cầu

- **PHP** 8.2 trở lên
- **Composer** 2.x
- **Laravel** 11.x
- **MySQL** 8.0 / MariaDB 10+
- **Node.js** 20+ (dành cho frontend)
- **Redis** (tùy chọn, cho cache & session)

### Cài đặt nhanh

```bash
# 1. Clone repository
git clone https://github.com/AbdolHamidDev/Mobifone.git

# 2. Di chuyển vào thư mục dự án
cd Mobifone

# 3. Cài đặt các phụ thuộc PHP
composer install

# 4. Sao chép file cấu hình môi trường
cp .env.example .env

# 5. Cấu hình file .env (database, mail, pusher, redis...)
# 6. Tạo khóa ứng dụng
php artisan key:generate

# 7. Chạy migration
php artisan migrate

# 8. Import dữ liệu mẫu
#    Import file SQL có sẵn vào database
#    📂 database/sample/mobifone.sql

# 9. Cài đặt frontend dependencies
npm install

# 10. Build frontend
npm run build

# 11. Khởi động server
php artisan serve
```

### ⚠️ Quan trọng: Import dữ liệu mẫu

**❗ Nếu không import cơ sở dữ liệu mẫu, ứng dụng sẽ bị lỗi do thiếu dữ liệu ràng buộc.**

📂 **Đường dẫn file SQL:** `database/sample/mobifone.sql`

#### Cách import vào PHPMyAdmin:
1. Mở **phpMyAdmin** tại `localhost/phpmyadmin`
2. Tạo cơ sở dữ liệu mới (ví dụ: `mobifone`)
3. Chọn database → tab **Import**
4. Chọn file `mobifone.sql` → nhấn **Go**

#### Hoặc import bằng command line:
```bash
mysql -u root -p mobifone < database/sample/mobifone.sql
```

### Cấu hình môi trường (.env)

```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=mobifone
DB_USERNAME=root
DB_PASSWORD=

# Pusher (chat real-time)
PUSHER_APP_ID=your_app_id
PUSHER_APP_KEY=your_app_key
PUSHER_APP_SECRET=your_app_secret
PUSHER_APP_CLUSTER=ap1
BROADCAST_DRIVER=pusher

# Redis (tùy chọn)
REDIS_HOST=127.0.0.1
REDIS_PASSWORD=null
REDIS_PORT=6379
```

### Chạy ứng dụng trong môi trường phát triển

```bash
# Chạy đồng thời: Laravel server + Queue worker + Log viewer + Vite dev
composer run dev
```

Hoặc chạy riêng lẻ:
```bash
# Laravel server
php artisan serve

# Queue worker (chat, email...)
php artisan queue:listen --tries=1

# Vite dev server (hot reload)
npm run dev
```

Chúng tôi hoan nghênh mọi đóng góp từ cộng đồng! Vui lòng tham khảo:

1. **Fork** dự án về tài khoản của bạn
2. Tạo branch feature mới: `git checkout -b feature/amazing-feature`
3. **Commit** thay đổi: `git commit -m 'Add amazing feature'`
4. **Push** lên branch: `git push origin feature/amazing-feature`
5. Tạo **Pull Request** (PR)
